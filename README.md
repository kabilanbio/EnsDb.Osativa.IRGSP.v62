# EnsDb.Osativa.IRGSP.62
Provides the Ensembl Plants gene, transcript, and exon annotations     for the rice genome assembly IRGSP-1.0, based on the official Ensembl     release 62. Gene IDs follow the RAP-DB nomenclature (OsXXgXXXXXXX) and     are complemented with MSU v7.0 functional descriptions where available.
